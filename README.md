# University Management System API

Backend API built with ASP.NET Core using a clean architecture (3-Layer: API / Business / Data).

##  Overview

This project is a backend API for managing a university system, with features for:
- Users authentication using JWT
- Role based access (Admin / User)
- Permission based authorization
- Managing courses, students, instructors and their relations

##  Features

- JWT Authentication (Login / Register)  
- Role & Permission Authorization  
- Secure Password Hashing  
- Entity Framework Core with SQL Server  
- Swagger UI for API testing  

##  Tech Stack

- **Language:** C#  
- **Framework:** ASP.NET Core  
- **ORM:** Entity Framework Core  
- **Authentication:** JWT  
- **Database:** SQL Server  
- **Documentation:** Swagger

##SetUp

1. Clone the repo
2. Create appsettings.json from appsettings.Development
3. Run migrations or database setup
4. Run project in Visual Studio
